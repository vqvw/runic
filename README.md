# Runic Parser

This repository contains the Runic lexer + parser + transpiler.

Other Runic repositories are located at:

- Runic Web: [https://github.com/vqvw/runic_web](https://github.com/adambarjo/runic_web)
- Runic CLI: [https://github.com/vqvw/runic_cli](https://github.com/adambarjo/runic_cli)
- Runic WASM: [https://github.com/vqvw/runic_wasm](https://github.com/adambarjo/runic_wasm)

## Installation

`go get github.com/vqvw/runic`

## Requirements

- `go1.22.1`
